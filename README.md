# 🛍️ ModernShop Frontend — E-Commerce UI

A modern, fast, and scalable **E-Commerce Frontend** inspired by Fnac, built with **React 18**, **Vite**, and **Tailwind CSS**.

> ⚡ Optimized for conversion, performance, and seamless user experience.

---

## 🚀 Tech Stack

* ⚛️ React 18
* ⚡ Vite
* 🎨 Tailwind CSS 3
* 🔗 Axios (API calls)
* 🧠 Context API / Hooks
* 🧭 React Router v6
* ⭐ Rating UI Components

---

## ✨ Frontend Features

### 🔐 Authentication UI

* User registration & login forms
* Persistent authentication (token-based)
* Protected routes (user dashboard, checkout)
* Guest mode support

---

### 🏠 Homepage

* Hero banners / promotions
* Featured products section
* Categories highlights
* Trending & best-selling products

---

### 🔍 Product Discovery

* Advanced search bar
* Filters:

  * Category
  * Brand
  * Price range
  * Ratings
* Sorting:

  * Price (low → high / high → low)
  * Popularity
  * New arrivals

---

### 📦 Product Details Page

* Image gallery with preview
* Product variations (size, color…)
* Ratings & reviews display
* Stock availability indicator
* Add to cart / wishlist actions

---

### 🛒 Cart System

* Add / remove products
* Quantity management
* Real-time total calculation
* Persistent cart (guest + logged user)

---

### ❤️ Wishlist

* Save favorite products
* Move items to cart
* Sync with user account

---

### 💳 Checkout UI

* Multi-step checkout process:

  * Address selection / creation
  * Order summary
  * Payment method selection
* Validation & error handling

---

### 📋 User Dashboard

* Order history
* Order details & tracking UI
* Address management
* Profile settings

---

### 🏷️ Promotions & Discounts

* Promo code input UI
* Discount calculation preview

---

### ⭐ Reviews & Ratings

* Add review form
* Star rating system
* User feedback display

---

## 🧠 UX Features

* ⚡ Fast SPA navigation
* 📱 Fully responsive (mobile-first)
* 🧩 Reusable UI components
